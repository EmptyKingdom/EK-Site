== Description ==
A simple single post footer to include sharing links for your visitors. 

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the 'simple-social-sharing' folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Select the area(s) you want the sharing links to appear
1. That's it.

== Frequently Asked Questions ==

= What does this do? =

It allows people to easily share your content over the social web without a lot of overhead.
